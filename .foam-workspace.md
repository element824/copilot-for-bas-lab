# Foam workspace marker

This empty file tells the **Foam** extension that this folder is a wiki workspace.

With Foam installed (see `.vscode/extensions.json`) the following Confluence-like behaviour is enabled across every `.md` file in this repo:

- **`[[Page Name]]` wiki links** — type `[[` and you get autocomplete for every page in the repo.
- **Backlinks panel** — every page shows "Linked references" (which other pages link to this one).
- **Graph view** — visualise the whole space as a node graph (`Foam: Show Graph` from the Command Palette).
- **Daily notes / journal** — `Foam: Open Daily Note` for meeting notes, BA workshops, etc.

> 💡 For the demo: open Foam's graph view and show the BAs how every story, persona and ADR is connected. This is the **"Confluence space map, but live"** moment.

Do not delete this file — Foam uses its presence to detect the workspace root.
